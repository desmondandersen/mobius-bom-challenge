# Mobius Materials BOM Challenge

Basic front-end application that allows a user to fetch a bill of materials. Created for Mobius Materials coding challenge.

### Local Testing

Make sure you have the latest versions of node and npm installed.

1. Clone repository and navigate to project directory.
2. Run `npm install`.
3. Use `yarn start` to start the app in the development mode. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
